# smart-todo-app
This is a simple To-Do List web application built using HTML, CSS, and JavaScript. The project helps users manage daily tasks by adding, deleting, and marking them as completed. It uses localStorage to save tasks so data remains even after refreshing the page. This project was created to practice DOM manipulation and JavaScript fundamentals.
