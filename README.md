# smart-todo-app
This is a simple To-Do List web application built using HTML, CSS, and JavaScript. The project helps users manage daily tasks by adding, deleting, and marking them as completed. It uses localStorage to save tasks so data remains even after refreshing the page. This project was created to practice DOM manipulation and JavaScript fundamentals.
Deployed Link: https://effervescent-buttercream-a0fbf1.netlify.app/
Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All / Completed / Pending)
- Data stored using localStorage

 What I Learned
- DOM manipulation
- Event handling
- Working with localStorage
- Building interactive UI without any framework

 Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)
- Browser localStorage API
